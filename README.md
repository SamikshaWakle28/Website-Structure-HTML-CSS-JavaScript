# Website-Structure-HTML-CSS-JavaScript

## Core Technologies
HTML5: Semantic structure for optimal SEO and accessibility.
CSS3: Modern Glassmorphism UI, CSS Grid/Flexbox layouts, and scroll-triggered animations.
JavaScript : Custom-built Digital Assistant (Chatbot), dynamic skill progress bars, and responsive navigation logic.

## Key Features
Responsive Bento-Grid: Adaptive Services and Gallery sections that optimize for Desktop, Tablet, and Mobile.
Interactive UI: Scroll-animated expertise bars and hover-active project overlays.
Smart Assistant: A lightweight JS-driven chatbot designed for instant SME lead generation and support.
Direct Connect: Integrated WhatsApp and Email CTA triggers for seamless client communication.
